02. Introducing GreenKode

GreenKode is a professional CLI tool that empowers developers to measure, analyze, and optimize the carbon footprint of their code.

user@greenkode:~/project

Static Analysis

Scans source code patterns (AST) to detect algorithmic inefficiencies like $O(n^2)$ loops before execution.

Live Auditing

Measures real-time power draw using Intel RAPL sensors and converts it to CO2 equivalents.

GreenKode Proposal Page 3

03. Under the Hood

Bridging high-level Python code with low-level hardware sensors.

Hardware Integration

GreenKode interfaces directly with Intel RAPL registers to read energy consumption in microjoules. This provides lab-grade accuracy without external hardware meters.

04. The Experiment

Validating GreenKode by comparing inefficient vs. optimized code.

Methodology

Task: Find duplicate integers in N=5,000 dataset.

# Dirty Code: O(n^2)
for i in range(len(data)):
  for j in range(i+1, len(data)):
    if data[i] == data[j]:
      duplicates.append(data[i])
# Green Code: O(n)
seen = set()
for item in data:
  if item in seen:
    duplicates.add(item)
  else:
    seen.add(item)
